
# نصب گیت از سورس در Rocky Linux 9

### مرحله 1: دانلود سورس گیت

برای شروع، سورس گیت نسخه 2.52.0 را با استفاده از دستور `wget` دانلود کنید:

```bash
wget https://www.kernel.org/pub/software/scm/git/git-2.52.0.tar.gz
```

### مرحله 2: استخراج فایل آرشیو

پس از دانلود، فایل آرشیو را با استفاده از دستور زیر استخراج کنید:

```bash
tar -xzf git-2.52.0.tar.gz
```

### مرحله 3: وارد دایرکتوری سورس گیت شوید

وارد دایرکتوری سورس گیت استخراج شده شوید:

```bash
cd git-2.52.0/
```

### مرحله 4: نصب وابستگی‌های لازم

قبل از ساخت گیت، وابستگی‌های لازم را با استفاده از دستور `yum` نصب کنید:

```bash
yum install curl-devel expat-devel gettext-devel openssl-devel zlib-devel perl-CPAN perl-devel -y
```

### مرحله 5: کامپایل گیت

پس از نصب وابستگی‌ها، گیت را با استفاده از دستور `make` کامپایل کنید. پیش‌فرض نصب را روی `/usr/local` تنظیم کنید:

```bash
make prefix=/usr/local all
```

### مرحله 6: نصب گیت

پس از اتمام فرایند کامپایل، گیت را روی سیستم نصب کنید:

```bash
make prefix=/usr/local install
```

### مرحله 7: بررسی نصب

پس از نصب، بررسی کنید که گیت به درستی نصب شده است با دستور زیر:

```bash
git --version
```

باید خروجی مشابه زیر را مشاهده کنید:

```bash
git version 2.52.0
